# Downstream Deck Review Rubric

Use this after NotebookLM has generated a deck.

## Review questions

1. Does the deck follow the intended teaching story rather than paper section order?
2. Does `Required coverage` appear in recognizable slide form?
3. Did NotebookLM drift back into abstract-summary slides?
4. Are the slides visually aligned with the intended style?
   - white background
   - black text
   - colorful figures or diagrams where helpful
   - clean academic look
5. Are the slides reasonably low-density?
6. Do figures and diagrams help explain the core ideas?

## Outcome labels

- **Pass** — deck is aligned enough to use with light edits
- **Revise prompt** — deck misses structure or style; adjust prompt/template
- **Revise story** — upstream teaching narrative or coverage list is weak
