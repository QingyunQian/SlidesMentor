# Codex Plugin Entry

This directory is the root Codex-facing entrypoint for this repository.

- [plugin.json](plugin.json) provides plugin metadata.
- [install.sh](install.sh) syncs the canonical skill into a Codex skills directory.

Canonical skill content still lives under [slidesmentor-skill-package/](../slidesmentor-skill-package). The root [skills/](../skills) directory is present for Codex plugin discovery.
